*   `command_exit_zero`: Checks if a shell command exited successfully (return code 0).
*   `file_exists`: Checks if a file exists at a given path.
*   `text_in_file`: Checks if a specific string is present in a file.
*   `api_status_ok`: Checks if an API call returned a successful status code (2xx).
